Initial setup
    default csv file name :- question.csv
    default timeout intercal:- 30 sec

To change or pass custom filename, you need to use fname flag
    ex go run quiz.go -fname <path to csv filename>

To change or pass custom time-interval, you need to use timeout-interval flag  
    ex go run quiz.go -fname <path to csv filename> -timeout-interval <time in seconds>
